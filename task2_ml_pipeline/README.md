# Task 2: Customer Churn Prediction ML Pipeline

## Objective
Build a production-ready ML pipeline to predict customer churn.

## Dataset
Telco Customer Churn Dataset (IBM)

## Methodology
- Preprocessed data (encoding, scaling, null handling)
- Built Pipeline with StandardScaler + models
- Trained Logistic Regression and Random Forest
- Used GridSearchCV for hyperparameter tuning
- Exported best model using joblib

## Results
- Logistic Regression Accuracy: ~80%
- Random Forest Accuracy: ~82%

## Tools Used
- scikit-learn, pandas, joblib, Streamlit
